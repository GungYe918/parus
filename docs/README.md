# Parus Documentation

Parus 문서는 목적별로 다음 4계층으로 분리한다.

1. `reference/`: 언어/ABI 정본 문서 (Normative)
2. `guides/`: 사용자 안내 문서
3. `internal/`: 컴파일러 개발자 내부 문서
4. `architecture/`: 언어 철학/중장기 설계 문서

우선순위 규칙:

1. 주제별 `Normative` 문서
2. `docs/reference/language/SPEC.md`
3. Guide/Architecture/Internal 문서

빠른 시작:

1. 언어 정본 인덱스: `docs/reference/language/SPEC.md`
2. ABI 정본 인덱스: `docs/reference/abi/README.md`
3. 사용자 가이드: `docs/guides/user/manual_v0.md`
4. 컴파일러 내부 문서: `docs/internal/compiler/README.md`
