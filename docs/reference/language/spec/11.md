## 7.2 switch

### 문법

```parus
switch (expr) {
  case LIT: { ... }
  case LIT: { ... }
  default: { ... }
}
```

### 규칙

* fallthrough 금지
* `case` 라벨은 리터럴만 허용
  (int, string, bool, char 등)
* `default`는 선택 사항이나 강력히 권장
* 타입 패턴 매칭 허용

  ```parus
  case Err(name): { ... }
  ```

  * `name`은 새 바인딩이며, 스코프는 해당 `case` 블록 내부

### 예시

```parus
def demo(r: Result<u32>) -> u32 {
  switch (r) {
    case Ok(v): { return v; }
    case Err(e): {
      // e: Error
      return 0u32;
    }
  }
}
```
